Webcomponent to allow users download multiple logotype image types when they trying to grab low resolution logo.
  • Demo
  • Usage
  • About

Right-click the logo above to see the example.

The web component checks for <link> tags in the header with type=image/extension attribute and then creates links, so the user in a easy way can download the high resolution vector logo for example.
