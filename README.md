# \<logo-downloadtip>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i logo-downloadtip
```

## Usage
```html
<script type="module">
  import 'logo-downloadtip/logo-downloadtip.js';
</script>

<logo-downloadtip></logo-downloadtip>
```
